## 安装

```bash
npm install --save-dev raw-loader
```

## 用法

通过 webpack 配置、命令行或者内联使用加载器。

### 通过 webpack 配置（推荐）

**webpack.config.js**
```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.txt$/,
        use: 'raw-loader'
      }
    ]
  }
}
```

**在你的项目中**
```js
import txt from 'file.txt';
```

### 通过命令行（CLI）

```bash
webpack --module-bind 'txt=raw-loader'
```

**在你的项目中**
```js
import txt from 'file.txt';
```

### 内联使用

**在你的项目中**
```js
import txt from 'raw-loader!./file.txt';
```
