# script loader for webpack

## 用法

``` javascript
require("script-loader!./file.js");
// => 在全局上下文执行了一次 file.js 
```

[文档：使用加载器](http://webpack.github.io/docs/using-loaders.html)

在 node.js 中不做任何事。
