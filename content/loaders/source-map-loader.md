# webpack的source map loader

为原文件提取SourceMaps作为 `sourceMappingURL` 的注释。

## 使用

[文档：使用loaders](https://webpack.js.org/concepts/#loaders)

### webpack配置示例

``` javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: ["source-map-loader"],
        enforce: "pre"
      }
    ]
  }
};
```

这将从所有文件中提取所有SourceMaps。这不是那么的性能，所以你可能只想应用加载器到相关的文件。
